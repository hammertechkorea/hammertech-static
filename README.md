# 홈페이지 수정 시작 가이드

우리 홈페이지([www.hammertech.co.kr](http://www.hammertech.co.kr))를 직접 고치고 인터넷에 올리는 방법입니다. 어렵지 않아요. 처음 한 번만 준비하면, 그다음부턴 5줄 복사-붙여넣기가 전부예요.

---

## 📌 이게 무슨 원리인가요? (30초)

우리 홈페이지 파일들은 **GitHub**라는 인터넷 창고에 들어 있어요.  
그걸 내 컴퓨터로 가져와서 → 고치고 → 다시 올리면 → 홈페이지에 자동으로 반영됩니다.  
올리는 동작을 **"push(푸시)"** 라고 불러요. 이 단어 하나만 기억하면 됩니다.

---

## A. 처음 딱 한 번만 하는 준비

### 1) 프로그램 설치하기
아래 2개를 받아서 설치합니다. (둘 다 "다음 → 다음" 클릭만 하면 돼요)

* **Git**: [다운로드 링크](https://git-scm.com/download/win)
  * 설치하면 "Git Bash" 라는 검은 창 앱이 같이 깔립니다. 앞으로 명령어는 이 Git Bash 창에 입력해요.
* **VS Code** (글·사진을 고치는 프로그램): [다운로드 링크](https://code.visualstudio.com)

> [!TIP]
> 시작 메뉴에서 "Git Bash" 를 검색하면 검은 창이 열립니다. 이 창이 우리 작업 창이에요.

### 2) 로그인하기 (한 번만)
Git Bash 창에 아래 한 줄을 입력하고 **Enter**:

```bash
gh auth login
```

화면 안내를 따라가면 브라우저가 열립니다. 우리 GitHub 계정으로 로그인하세요. (한 번 로그인하면 컴퓨터가 기억하므로 다시 안 해도 됩니다.)

### 3) 홈페이지 파일 내려받기 (한 번만)
1. **I 드라이브**에 `홈페이지` 폴더를 만듭니다. (이미 폴더가 있다면 건너뛰셔도 됩니다.)
2. Git Bash 창에 아래 두 줄을 차례로 입력하고 **Enter**:

```bash
cd /i/홈페이지
git clone https://github.com/hammertechkorea/hammertech-static.git 해머텍코리아
```
→ `I:\홈페이지\해머텍코리아` 폴더가 생성되며 홈페이지 파일이 다운로드됩니다.

---

> [!NOTE]
> **준비 끝!** 2)번과 3)번은 앞으로 다시 할 필요 없습니다.

---

## B. 그다음부터 — 홈페이지 고칠 때마다

### 1) 내용 고치기
1. **VS Code** 를 열고 → **File** → **Open Folder** → `I:\홈페이지\해머텍코리아` 폴더 선택
2. 원하는 파일을 클릭해 글·사진을 고친 뒤 **Ctrl + S** 로 저장

### 2) 인터넷에 올리기
Git Bash 창에 아래 5줄을 위에서부터 차례로 복사-붙여넣기 하세요:

```bash
cd /i/홈페이지/해머텍코리아
git pull
git add .
git commit -m "수정함"
git push
```

#### 각 줄의 뜻:

| 명령어 | 하는 일 |
| :--- | :--- |
| `cd /i/홈페이지/해머텍코리아` | 홈페이지 폴더 안으로 들어가기 |
| `git pull` | 최신 버전 받아오기 |
| `git add .` | 고친 내용 담기 |
| `git commit -m "수정함"` | "고쳤다" 도장 찍기 (따옴표 안 글자는 자유롭게) |
| `git push` | 인터넷에 올리기 |

* `git push` 후 약 1분 뒤 홈페이지에 반영됩니다.
* 홈페이지에서 확인할 때는 **Ctrl + Shift + R** (강력 새로고침)을 눌러주세요.

---

> [!NOTE]
> **끝!** 앞으로는 이 5줄만 반복하면 됩니다.

## ❓ 자주 묻는 질문

**Q. 잘못 고쳐서 홈페이지가 이상해졌어요.**  
A. 당황하지 마세요. 이전 버전이 GitHub에 전부 남아 있어 되돌릴 수 있습니다. 관리자에게 알려주세요.

**Q. git push 했는데 홈페이지가 그대로예요.**  
A. 1~2분 기다린 뒤 **Ctrl + Shift + R** 로 새로고침 해보세요.

**Q. 명령어 입력 중 에러 글자가 떴어요.**  
A. 그 화면을 캡처해서 관리자에게 보내주세요.
